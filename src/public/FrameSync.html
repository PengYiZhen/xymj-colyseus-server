📖 使用说明

1. 连接房间：输入服务器地址和房间名称，点击"连接房间"按钮。

2. 发送输入：使用方向键按钮或自定义输入发送游戏输入。

3. 查看状态：实时查看连接状态、当前帧号等信息。

4. 监控日志：查看所有操作和消息的日志记录。

注意：确保服务器正在运行，并且已安装 Colyseus 客户端库。 如果使用浏览器，需要先安装：npm install colyseus.js
